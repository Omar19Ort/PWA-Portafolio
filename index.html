Proyectos Destacados

Memorama

Un juego de memoria con imágenes de personajes de caricatura, con varios modos de dificultad. Ingresas con tu nombre y tiene cronómetro para registrar el tiempo. Hecha con JavaScript, HTML y CSS.

TECH-OM

APK desarrollada en Flutter y Dart para la gestión de reparaciones de celulares y computadoras. Controla cada cotización de reparaciones, además del registro de usuarios.

Página web para el control de centro canino, donde puedes registrar a tu mascota, pagar por servicios, y comprar juguetes y comida para tu mascota. Hecha con PHP, HTML y CSS.
